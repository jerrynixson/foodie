# Foodie

Minimal personal nutrition helper (backend + Streamlit pages).

Quick start (Windows):

1. Clone the repo:

	git clone https://github.com/jerrynixson/foodie.git

2. Install the helper tool and create a virtual environment:

	pip install uv
	uv sync

3. Activate the virtual environment (Windows):

	venv\Scripts\activate

4. Run the app:

	python run.py

Notes:
- Source code lives under `src/foodie`.
- rerun the app (python run.py) after any change to code as currently there is no hot reload
